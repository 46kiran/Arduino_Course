Module 1: Introduction to Arduino & Basic Electronics

‚ñº

Week 1: Getting Started

  • What is Arduino? Understanding microcontrollers
  • Arduino board anatomy (Uno, Nano, Mega)
  • Setting up Arduino IDE
  • Basic electronics: voltage, current, resistance

Challenge 1.1: First Blink

Write code to make the onboard LED blink at different speeds. Modify the delay to create a custom pattern (3 fast blinks, 1 slow blink, repeat).

Learning goal: Understand digitalWrite() and delay() functions

Challenge 1.2: Build Your First Circuit

Connect an external LED with a resistor on a breadboard. Calculate the correct resistor value for your LED. Make it blink in Morse code to spell "SOS".

Learning goal: Apply Ohm's Law, understand breadboard connections

Module 1 Project: Traffic Light Simulator

Build a 3-LED traffic light system that cycles through red, yellow, and green with realistic timing. Add a pedestrian crossing button that interrupts the cycle.

Skills practiced: Multiple outputs, timing control, basic input handling

Module 2: Digital & Analog I/O

‚ñº

Week 2: Working with Inputs and Outputs

  • Digital vs. Analog signals
  • Using buttons and switches (digitalRead)
  • Pull-up and pull-down resistors
  • Reading analog sensors (analogRead)
  • PWM (Pulse Width Modulation) for analog output
  • Serial communication basics
Challenge 2.1: Button-Controlled LED

Create a system where pressing a button toggles an LED on/off (not just while pressed). Add debouncing to prevent false triggers.

Learning goal: State management, debouncing techniques

Challenge 2.2: Potentiometer Dimmer

Use a potentiometer to control LED brightness smoothly. Display the brightness level (0-100%) on Serial Monitor.

Learning goal: Analog input, PWM output, map() function

Challenge 2.3: Light-Reactive Nightlight

Use a photoresistor (LDR) to automatically turn on an LED when it gets dark. Make the LED brightness inversely proportional to ambient light.

Learning goal: Sensor integration, conditional logic

Module 2 Project: Digital Piano

Build a 4-key piano using buttons that play different tones through a buzzer. Add a potentiometer to control the octave. Display the note being played on Serial Monitor.

Skills practiced: Multiple inputs, tone generation, PWM, serial output

Module 3: Sensors & Data Processing

‚ñº

Week 3-4: Working with Sensors

  • Temperature sensors (LM35, DHT11)
  • Distance measurement (Ultrasonic sensor)
  • Motion detection (PIR sensor)
  • Data smoothing and filtering
  • Calibration techniques
  • Arrays and data storage
Challenge 3.1: Temperature Monitor

Read temperature from a sensor and display it on Serial Monitor. Trigger different LEDs for cold (<20¬∞C), comfortable (20-25¬∞C), and hot (>25¬∞C) ranges.

Learning goal: Sensor reading, conditional logic, thresholds

Challenge 3.2: Parking Sensor

Use an ultrasonic sensor to create a parking assistant. Beep faster as objects get closer. Add LED indicators for distance ranges.

Learning goal: Distance calculation, variable timing, multi-output control

Challenge 3.3: Data Logger

Record temperature readings every second for 1 minute. Calculate and display the average, minimum, and maximum values.

Learning goal: Arrays, loops, mathematical operations

Module 3 Project: Smart Room Monitor

Combine temperature, light, and motion sensors to create an intelligent room monitoring system. Display status on Serial Monitor. Turn on lights when motion is detected and it's dark, and trigger an alert when temperature exceeds limits.

Skills practiced: Multiple sensor integration, complex logic, system design

Module 4: Displays & Visual Feedback

‚ñº

Week 5: Output Devices

  • LCD displays (16x2)
  • 7-segment displays
  • LED matrices
  • RGB LEDs
  • Using libraries effectively
  • Creating custom characters
Challenge 4.1: LCD Thermometer

Display temperature readings on an LCD screen with proper formatting. Add a custom degree symbol. Update the display every 2 seconds.

Learning goal: LCD library usage, formatting, custom characters

Challenge 4.2: Countdown Timer

Create a countdown timer displayed on a 7-segment display. Use buttons to set minutes and seconds, and another button to start/stop.

Learning goal: Display multiplexing, time management, user interface

Challenge 4.3: Mood Lamp

Use an RGB LED to create a color-changing mood lamp. Implement smooth color transitions. Add buttons or a potentiometer to control speed and color selection.

Learning goal: RGB mixing, color theory, smooth transitions

Module 4 Project: Digital Clock with Alarm

Build a digital clock using an LCD display. Include settable alarm functionality with buttons. Use a buzzer for the alarm sound. Display temperature as a bonus feature.

Skills practiced: Time keeping, user interface design, display management, integrated systems

Module 5: Motors & Motion Control

‚ñº

Week 6: Actuators and Movement

  • DC motors and H-bridge control
  • Servo motors (position control)
  • Stepper motors (precise movement)
  • Motor drivers (L298N, L293D)
  • Power management for motors
  • Speed control with PWM
Challenge 5.1: Servo Sweep

Control a servo motor to sweep back and forth. Add a potentiometer to manually set the servo position. Display the angle on Serial Monitor.

Learning goal: Servo library, angle control, input-to-output mapping

Challenge 5.2: DC Motor Speed Control

Control DC motor speed and direction using buttons or a potentiometer. Implement acceleration/deceleration rather than instant speed changes.

Learning goal: Motor driver usage, PWM for speed, direction control

Challenge 5.3: Stepper Motor Precision

Move a stepper motor exactly 360 degrees, pause, then return to start position. Implement speed control. Count and display total steps taken.

Learning goal: Stepper motor control, precision movement, step counting

Module 5 Project: Robotic Arm or Line-Following Robot

Option A: Build a 2-axis robotic arm controlled by potentiometers. Include a servo-controlled gripper.
Option B: Create a simple line-following robot using IR sensors and DC motors.

Skills practiced: Motor integration, sensor-motor coordination, mechanical design

Challenge 6.2: Temperature Data Logger

Log temperature readings to an SD card with timestamps. Retrieve and display the data when requested via Serial Monitor.

Learning goal: SD card operations, data formatting, file handling

Challenge 6.3: I2C Sensor Network

Connect multiple I2C sensors (temperature, pressure, light) and display all readings on an LCD screen. Cycle through different sensor displays.

Learning goal: I2C protocol, multiple device addressing, display management
